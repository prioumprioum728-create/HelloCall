HelloCall Firebase Functions

This page explains the Firebase functionality used in the HelloCall project.

Simulated Firebase Features

  • Push call logs to Firebase (simulation only)
  • Store call history in Firebase Realtime Database
  • Simulated user authentication (no real credentials)
  • Safe testing environment for learning

Related Files

  • publish_firebase_o2tt.py – Python simulation of Firebase publishing
  • firebase.html – Firebase setup documentation
  • firebase_service.html – Additional Firebase info
  • services.json – Lists app services connecting to Firebase

Notes

All Firebase operations in HelloCall are simulated and safe. No private credentials are included.
