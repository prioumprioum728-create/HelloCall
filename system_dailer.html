HelloCall System Dialer

This page documents the system dialer functionality for the HelloCall app.

App Feature

  • Simulates phone calls for testing and demonstration purposes
  • Call logs are saved in Firebase Realtime Database
  • Uses bundle ID: com.hellocall.hellocall
  • Safe, isolated, does not access real device dialer

Firebase Integration

Call logs are pushed to Firebase using the hellocall_firebase_service.json credentials file.

Usage

  1. Run hellocall.py in a Python environment (Cloud Shell or local)
  2. Enter caller name, callee number, and call duration
  3. Call log will be added to Firebase and displayed

Notes

  • Static HTML file for documentation only
  • Safe to upload to GitHub
  • Does not actually make phone calls
