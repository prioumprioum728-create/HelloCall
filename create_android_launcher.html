Create Android Launcher for HelloCall

This page documents the steps to create an Android launcher for the HelloCall app.

1. Set up Android Project

  • Use Android Studio or a compatible tool
  • Create a new project with package name: com.hellocall.hellocall
  • Choose Launcher Activity as the main entry point

2. Integrate HelloCall Backend

  • Use Firebase Realtime Database
  • Include hellocall_firebase_service.json credentials (keep private)
  • Add call log functionality using Android SDK

3. Create App Icons and UI

  • Add launcher icons in res/mipmap folders
  • Customize layout for the main activity
