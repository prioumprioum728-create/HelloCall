# HelloCall Wiki

Welcome to the HelloCall project wiki! This file explains how the app works and its features.

## Overview
HelloCall is a safe phone call simulation project for learning and documentation.  
It includes:
- System dialer simulation
- Firebase call log simulation
- Internal storage simulation
- HTML and Markdown documentation

## File Structure
- **README.md**: Project overview  
- **LICENSE.txt**: MIT license  
- **services.json**: App services/features info  
- **firebase.html**: Firebase setup documentation  
- **system_dailer.html**: System dialer documentation  
- **create_android_launcher.html**: Launcher guide  
- **system.md**: System documentation in Markdown  
- **style.css**: Styling for HTML files  
- **publish_firebase_o2tt.py**: Safe Python placeholder  
- **hello_sys.sys, system.csw, debug.sys, win100.bin, install.wim, wim.sys**: Safe placeholder system files  

## How to Use
1. Open `.html` files in a browser to view documentation.  
2. Open `.py` files with Python on your PC to simulate functionality.  
3. All `.sys`, `.bin`, `.wim`, `.csw` files are placeholders for structure only.  
4. All files are safe and can be uploaded to GitHub for learning purposes.

## Notes
- This wiki is contained in **one file**: `wiki.md`  
- No real phone calls or Firebase credentials are included  
- Ideal for sharing your HelloCall project safely
